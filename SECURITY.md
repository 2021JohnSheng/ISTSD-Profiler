# Security Policy

ISTSD-Profiler is research software for local analysis of FASTA input files.

## Reporting a vulnerability

Please report security concerns by email instead of opening a public issue:

- Yong Sheng
- `johnsheng.sjtu@vip.163.com`

Include a short description of the issue, the affected version or commit, and
steps to reproduce when possible. Please avoid sharing sensitive input data in
public reports.
